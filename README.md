# OrgoLife - Organ Donation Platform

## 📌 Project Overview
OrgoLife is a full-stack web application designed to connect organ donors and recipients efficiently.

## 🚀 Features
- Donor Registration
- Recipient Requests
- Secure Authentication (JWT)
- API with FastAPI
- Responsive Frontend UI

## 🛠 Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: FastAPI (Python)
- Database: SQLite / Supabase
- Tools: Git, GitHub, Ngrok

## 📂 Project Structure
- /backend
- /frontend
- /database

## 💡 Future Improvements
- Real-time notifications
- Admin dashboard
- AI-based donor matching
